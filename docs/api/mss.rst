Model Structure Selection
-------------------------

These modules detect an optimal number of neurons in ELM, and remove extra unnecessary neurons.
They are used in ``ELM.train()`` if you give optional flag
``V`` (validation), ``CV`` (cross-validation) or ``LOO`` (Leave-One-Out validation).
